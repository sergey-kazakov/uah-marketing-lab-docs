# Visión general del proyecto

El ciclo de aplicaciones interactivas UAH Marketing Lab incluye entre 10 y 15 herramientas web diseñadas para apoyar la docencia en asignaturas de marketing: estratégico, comportamiento del consumidor, brand management, marketing digital, servicios y marketing internacional.

Cada aplicación funciona como un mini-laboratorio con datos simulados o reales, visualizaciones y reportes exportables.
