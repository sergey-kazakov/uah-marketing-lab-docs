# Manual Técnico · UAH Marketing Lab
### Ciclo de aplicaciones interactivas para la docencia en marketing (2025–2027)

Autores:
- **Dr. Pedro Cuesta Valiño**
- **Dr. Sergey Kazakov**
- **Dra. Patricia Durán Álamo**

Este manual describe la arquitectura técnica, los métodos analíticos, los datasets simulados y la documentación funcional de las aplicaciones interactivas desarrolladas para la enseñanza del marketing en la Universidad de Alcalá.
