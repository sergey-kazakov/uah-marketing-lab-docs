# UAH Marketing Lab · Technical Manual (Docusaurus)

Este proyecto contiene el Manual Técnico para el ciclo de aplicaciones interactivas de marketing (UAH Marketing Lab), en formato Docusaurus, con documentación bilingüe (español / inglés).

## Cómo usar

1. Instalar dependencias:

   npm install

2. Arrancar en modo desarrollo:

   npm start

3. Abrir en el navegador:

   http://localhost:3000

La documentación principal se encuentra en la carpeta `docs/es` y `docs/en`.
